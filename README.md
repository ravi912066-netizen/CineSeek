# 🎬 Movie App (Netflix Style UI)

A simple and clean movie web app built using **HTML, CSS, and JavaScript**.
It shows movies with posters, ratings, and details using API.

---

## 🚀 Features

* 🔍 Search Movies
* 🎥 Movie Cards (Poster, Title, Rating)
* ⭐ Add to Favorites (UI)
* 📱 Responsive Design
* 🌙 Dark Theme (Netflix style)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Movie API (like OMDb / TMDB)
